StockNova — Inventory Management System (Neon UI Prototype)

A modern, neon-styled Inventory Management System built using HTML, CSS, and JavaScript (LocalStorage).
This prototype includes Authentication, Products, Inventory, Categories, and Dashboard features — all running entirely in the browser.

 Features
Authentication System (LocalStorage)
Signup
Login
Password Reset
Input validation
User stored in browser
Product Management
Add, edit, delete products
Search and filter
LocalStorage product database

 Inventory Management

Add stock
Reduce stock
Track quantity
Low-stock alerts
Categories
Add categories
Assign categories to products
Filter by category

 Dashboard

Total products
Total stock
Low stock list
Recent activity

How to Run

Download or clone the project
Extract it anywhere
Open index.html in any browser (Chrome recommended)
Use the UI — everything runs locally
No backend or server required
